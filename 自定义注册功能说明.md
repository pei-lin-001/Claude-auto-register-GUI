# 🚀 自定义注册功能说明

## 功能概述

新增的自定义注册功能允许用户在仪表盘中灵活配置注册参数，支持单个账号注册和批量注册。

## 功能特点

### ✨ 主要特性

- **🎯 自定义注册数量** - 支持1-100个账号的批量注册
- **⚡ 并发控制** - 可设置1-10个并发浏览器窗口
- **⏰ 间隔控制** - 可设置5-3600秒的注册间隔时间
- **📊 实时预估** - 自动计算预计完成时间
- **🛡️ 参数验证** - 智能验证配置参数的合理性

### 🎨 界面设计

- **现代化UI** - Material Design 3.0风格
- **直观配置** - 清晰的参数说明和建议值
- **智能提示** - 详细的配置说明和注意事项
- **确认机制** - 双重确认避免误操作

## 使用方法

### 1. 启动自定义注册

1. 打开GUI应用程序
2. 在仪表盘中点击 **🚀 开始注册** 按钮
3. 弹出自定义注册配置对话框

### 2. 配置参数

#### 注册账号数量
- **范围**: 1-100个账号
- **建议**: 根据代理数量和时间安排合理设置
- **说明**: 设置为1时执行单个账号注册

#### 并发数量
- **范围**: 1-10个并发
- **建议**: 1-5个（避免资源占用过高）
- **说明**: 同时运行的浏览器窗口数量

#### 间隔时间
- **范围**: 5-3600秒
- **建议**: 60秒以上（避免频率过高）
- **说明**: 每个账号注册之间的等待时间

### 3. 确认执行

- 系统会显示配置摘要和预计时间
- 确认无误后点击确认开始注册
- 可随时在仪表盘中暂停/恢复/停止任务

## 配置建议

### 🎯 推荐配置

#### 小规模测试
```
账号数量: 1-3个
并发数量: 1个
间隔时间: 60秒
```

#### 中等规模注册
```
账号数量: 5-10个
并发数量: 2-3个
间隔时间: 120秒
```

#### 大规模批量注册
```
账号数量: 20-50个
并发数量: 3-5个
间隔时间: 180秒
```

### ⚠️ 注意事项

1. **代理质量** - 确保有足够的高质量代理IP
2. **网络稳定** - 保持网络连接稳定
3. **资源监控** - 注意CPU和内存使用情况
4. **频率控制** - 避免注册频率过高导致IP被封
5. **错误处理** - 及时查看日志处理异常情况
